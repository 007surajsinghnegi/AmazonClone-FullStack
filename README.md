# Installation Steps



## Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```

# Starting Stripe webhook
 
1) stripe listen  --forward localhost:3000/api/webhook

2) Check if the key provided matches the STRIPE_SIGNING_SECRET in env.local
